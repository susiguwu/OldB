## How to Enjoy OldBalance 

#1. 硬體/軟體準備
使用手機打開 https://susiguwu.github.io/OldB/
電腦打開地墊控制網頁 https://susiguwu.github.io/OldB/controller/index.html
(Group1判定為左邊，Group2中間，Group3右邊)

#2. 操作邏輯 (Mat Logic)
本遊戲透過 Firebase 即時接收地墊模擬器的訊號。請根據地墊的群組編號 (Group ID) 進行操作：
左邊水道: 踩踏地墊模擬器中的 Group 1。
中間水道: 踩踏地墊模擬器中的 Group 2。
右邊水道: 踩踏地墊模擬器中的 Group 3。

#3. 關卡說明 (Levels)
遊戲共分為三關，每關 20 秒，難度（下落速度）會隨關卡遞增：
Level 1: Bridge (獨木橋) - 保持平衡，穩定踩踏。
Level 2: Flowers (踩花) - 輕快反應，準確踩點。
Level 3: Logs (木頭) - 挑戰極限，快速移動。

#4. 計分方式
當物件下落至底部的淺黃色感應區時，踩下對應的地墊群組。
成功擊中一次得1分，遊戲結束後將顯示最終總分。
