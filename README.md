## How to Enjoy ★DIGOGO★

### 1. 硬體/軟體準備
* **手機端**：打開 [遊戲網頁](https://susiguwu.github.io/OldB/)
* **電腦端**：打開 [地墊控制網頁](https://susiguwu.github.io/OldB/controller/index.html)
* **注意**：Group 1 判定為左邊，Group 2 為中間，Group 3 為右邊。

### 2. 操作邏輯 (Mat Logic)
本遊戲透過 Firebase 即時接收地墊模擬器的訊號。請根據地墊的群組編號 (Group ID) 進行操作：
1. **左道**：踩踏地墊模擬器中的 Group 1。
2. **中間道**：踩踏地墊模擬器中的 Group 2。
3. **右道**：踩踏地墊模擬器中的 Group 3。

### 3. 關卡說明 (Levels)
遊戲共分為三關，每關 20 秒，難度（下落速度）會隨關卡遞增：
- **Level 1: 豬大哥接好運** - 踩踏豬大哥經過的方塊，沾沾喜氣。
- **Level 2: 借豬大哥過一下** - 踩踏豬大哥沒有經過的方塊。

### 4. 計分方式
- 當物件下落至底部的**感應區**時，踩下對應的地墊群組。
- 成功擊中/躲過一次得 **1分**，遊戲結束後將顯示最終總分。
